# Structure Data Status

This directory is reserved for structural inputs used in future versions of the project.

The current v1 workflow does not include a downloaded PDB or AlphaFold structure file. Instead, it uses an example PTEN residue-feature table containing structural-proxy values to demonstrate the scoring and visualisation pipeline.

The current repository should therefore be described as structure-informed, not as a fully structure-derived analysis.

Future versions should replace or supplement the example feature table with explicitly generated structural features, including solvent accessibility values calculated from a defined PTEN structure source such as PDB or AlphaFold.
